Redesign of Zappos - Springboard UX Design School Project

Zappos is known for providing an awesome customer experience, and in turn an awesome user experience. Despite Zappos' already great user experience, there are some aspects of the user experience that could be improved. In this project I identified and prototyped a few features - shopping profiles, cart reminders, running enthusiast product details, and a simplfied navigation scheme which prospectively improve the overall user experience.

The new features were tested with a small group of users, but additional testing is recommended for further validation.

Initial Analysis

In the initial analysis of the Zappos' website the SMART user experience strategy was employed to identify user and business goals for the home, product detail and shopping cart page. Additional features and restructing of some elements of the information architecture were also proposed.

Card Sorting - Navigation Scheme

Optimal Workshop's OptimalSort and Treejack where used for card sorting and reverse card sorting respectively. As can be seen from the tree analysis, similarity matrixand dendrogram, a revised top level navigation that based on gender rather than product type was supported. Additional user testing is recommended to help validate these results.
